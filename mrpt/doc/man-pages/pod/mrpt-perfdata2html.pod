=head1 NAME

mrpt-perfdata2html - Build HTML reports of performance tests

=head1 SYNOPSIS

mrpt-perfdata2html I<PATH_TO_PERFDATA_DIRECTORY>

=head1 DESCRIPTION

B<mrpt-perfdata2html> is a command-line tool for generating HTML reports and 
cross-build comparisons from all the found data generated by mrpt-performance
